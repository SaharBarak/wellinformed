What it does

Not another bookmarking tool. A live knowledge graph your agent queries autonomously.

MCP Native

11 tools your agent calls

search, ask, get_node, get_neighbors, find_tunnels, trigger_room, graph_stats, and more. Claude uses them without you copy-pasting anything.

Self-Indexing

Knows your codebase

Indexes .ts files, package.json deps, git submodules, commit history. Your agent searches your code and your research in one graph.

Active Fetching

8 source adapters

ArXiv API, Hacker News (Algolia), RSS/Atom feeds, any URL via Readability. Plus 4 project-indexing adapters. Background daemon keeps it fresh.

Rooms

Partitioned by domain

homelab, fundraise, ml-papers. Each room has its own sources and search scope. Tunnels surface when two rooms connect semantically.

Local

No cloud, no API keys

all-MiniLM-L6-v2 ONNX embeddings, sqlite-vec for search, NetworkX JSON for the graph. Everything under ~/.wellinformed/. Sub-100ms latency.

Architecture

Functional DDD

neverthrow Result monads, pure domain layer, ports + adapters infra, no classes. Every lib verified via ossinsight before selection.
